# LooLearn AI

LooLearn AI is a web application that simplifies trading and investment decisions by providing a suite of tools for backtesting, strategy development and visualization. Built with Streamlit, the app offers an intuitive interface for exploring financial data, testing trading strategies and understanding market behaviour.

## Features

- **Interactive dashboard:** Visualize historical price data, technical indicators and strategy performance in real time.
- **Strategy backtesting:** Quickly backtest trading strategies using customizable parameters.
- **Data exploration:** Pull and analyze data from sources like Yahoo Finance or cryptocurrency APIs.
- **User‑friendly interface:** Leverage Streamlit’s simple UI components to input ticker symbols, date ranges and strategy parameters.

## Technologies Used

- **Python** for data processing and back‑end logic.
- **Streamlit** for building the interactive web interface.
- **pandas** and **NumPy** for data manipulation and analysis.
- **matplotlib/Plotly** for data visualization.
- **yfinance** or other APIs for fetching historical market data.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/utkarshg20/loolearnai.git
   cd loolearnai
   ```
2. (Optional) Create and activate a virtual environment.
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Streamlit app:
   ```bash
   streamlit run apug.py
   ```
5. Open the provided local URL in your browser to interact with the app.
